# Education-Crisis-in-Kenya-From-Policy-to-Safety
This portfolio contains two end‑to‑end data analysis projects examining critical challenges in Kenya’s education sector. The first project evaluates the transition from the 8‑4‑4 system to the Competency‑Based Curriculum (CBC). The second project investigates the 2026 wave of secondary school unrest, including strikes, arson, and fatalities.
